# lima.studio — site estático

Este é um primeiro modelo de portfólio online, feito apenas com HTML, CSS e JavaScript.

## Como visualizar no computador

1. Baixe e extraia o arquivo ZIP.
2. Abra `index.html` no navegador.

## Como publicar gratuitamente

Opções possíveis:
- GitHub Pages
- Netlify
- Cloudflare Pages

Um endereço gratuito costuma ficar parecido com:
- `limastudio.github.io`
- `lima-studio.netlify.app`
- `lima-studio.pages.dev`

Para usar exatamente `lima.studio`, seria necessário comprar esse domínio.
